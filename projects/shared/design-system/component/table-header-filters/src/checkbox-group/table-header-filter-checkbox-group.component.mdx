# Table Header Filter Checkbox Group

This component is used to apply a filter to a table column using checkboxes.

This is a composite component using both the [Table Header Filter Wrapper](/story/components-table-header-filter-wrapper--docs) and the [Checkbox Group](/story/components-checkbox-group--docs) components so you will be able to use most of the inputs and outputs from both components.

Also, this component uses the [Table Header Filters Service](/story/services-table-header-filters--docs) to manage the filters and allow you to perform multi filtering.
