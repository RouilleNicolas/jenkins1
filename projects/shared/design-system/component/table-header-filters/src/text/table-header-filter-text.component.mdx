# Table Header Filter Text

This component is used to apply a filter to a table column using text input.

This is a composite component using both the [Table Header Filter Wrapper](/story/components-table-header-filter-wrapper--docs) and the [Material UI Input](https://material.angular.io/components/input/overview) components.

Also, this component uses the [Table Header Filters Service](/story/services-table-header-filters--docs) to manage the filters and allow you to perform multi filtering.
